# Kinesio Caseros - Landing Page

Landing page de alta conversión para kinesióloga en Caseros, Tres de Febrero.

## 🚀 Cómo usar

1. **Reemplazar número de WhatsApp**: En `script.js`, cambia `WHATSAPP_NUMBER` por tu número real (ej: `5491123456789`).
2. **Añadir fotos reales**: Sustituye los placeholders de imagen por fotos de la profesional atendiendo.
3. **Abrir en el navegador**: Abre `index.html` directamente o usa un servidor local.

## 📁 Estructura

- `index.html` - Contenido y estructura
- `styles.css` - Estilos mobile-first
- `script.js` - Micro-interacciones y formulario

## ✨ Características

- **Mobile First**: Optimizado para celulares (90% de pacientes)
- **Formulario simple**: 3 campos (Nombre, WhatsApp, Motivo)
- **Micro-interacciones**: Animaciones al hacer scroll
- **CTA**: Envío directo a WhatsApp
- **SEO local**: Palabras clave Caseros, Tres de Febrero, estación

## 🎨 Personalización

- Colores: `:root` en `styles.css`
- Fuentes: Google Fonts (DM Sans, Playfair Display)
